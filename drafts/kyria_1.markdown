## Combos

```
Top row
    Y + F        Z             W + :             !
C + Y         Q                    : + U         ?
C + Y + F      %               W + : + U         &

Home row
    T + H        Escape        N + A             One shot shift
S + T         Tab                  A + I           Enter
S +     H      -               N +     I          +
S + T + H      ;               N + A + I          _

Bottom row
    G + P     $               L + (             #
V + G         *                   ( + )         '
V +     P     ^
V + G + P     Movement layer  L + ( + )         Swedish layer
```
